# Reference: Resources, IDs, and Configuration Files

## Overview

This section documents the Win32 resources used by spisocketswin32: the application’s icons, menu definitions, keyboard accelerators, the About dialog, string table entries, and the corresponding identifiers defined in **resource.h**. These resources control the GUI elements (window icon, menu commands, dialogs) and allow customization of appearance and behavior without modifying C++ source code.

## Resource Script (spisocketswin32.rc)

**File:** spisocketswin32.rc

**Includes:**

- `"resource.h"`
- `"targetver.h"`
- `<windows.h>`

### Icons

```rc
// Icon with lowest ID value placed first to ensure application icon remains consistent on all systems.
IDI_SPIWAVWIN32 ICON    "spisocketswin32.ico"
IDI_SMALL         ICON    "small.ico"
```

- **IDI_SPIWAVWIN32**

– 32×32 application icon (`spisocketswin32.ico`).

- **IDI_SMALL**

– 16×16 icon used for small-window title bars (`small.ico`).

### Menu

```rc
IDC_SPIWAVWIN32 MENU
BEGIN
    POPUP "&File"
    BEGIN
        MENUITEM "E&xit",   IDM_EXIT
    END
    POPUP "&Help"
    BEGIN
        MENUITEM "&About .", IDM_ABOUT
    END
END
```

- **IDC_SPIWAVWIN32**

– Resource name for the menu.

- **File → Exit**

– Command identifier **IDM_EXIT**.

- **Help → About**

– Command identifier **IDM_ABOUT**.

### Accelerators

```rc
IDC_SPIWAVWIN32 ACCELERATORS
BEGIN
    "?",   IDM_ABOUT, ASCII, ALT
    "/",   IDM_ABOUT, ASCII, ALT
END
```

- Pressing **Alt + ?** or **Alt + /** invokes the About dialog (`IDM_ABOUT`).

### About Dialog (IDD_ABOUTBOX)

```rc
IDD_ABOUTBOX DIALOGEX 0, 0, 170, 62
STYLE DS_SETFONT | DS_MODALFRAME | DS_FIXEDSYS | WS_POPUP | WS_CAPTION | WS_SYSMENUCAPTION "About spiwavwin32"
FONT 8, "MS Shell Dlg", 0, 0, 0x1
BEGIN
    ICON        128, IDC_STATIC, 14,14,21,20
    LTEXT       "spiwavwin32, Version 1.0", IDC_STATIC, 42,14,114,8, SS_NOPREFIX
    LTEXT       "Copyright (C)\n2012",       IDC_STATIC, 42,26,114,8
    DEFPUSHBUTTON "OK", IDOK, 113,41,50,14, WS_GROUP
END
```

- **Dialog ID:** IDD_ABOUTBOX
- **Styles:** DS_SETFONT, DS_MODALFRAME, DS_FIXEDSYS, WS_POPUP, WS_CAPTION, WS_SYSMENUCAPTION
- **Font:** 8-pt MS Shell Dlg
- **Controls:**
- Static icon (at 14,14) bound to `IDC_STATIC`.
- Two lines of read-only text (`LTEXT`) sharing `IDC_STATIC`.
- Default “OK” button (`IDOK`) in the bottom-right.

### String Table

```rc
STRINGTABLE
BEGIN
    IDS_APP_TITLE      "spisocketswin32"
    IDC_SPIWAVWIN32    "SPISOCKETSWIN32"
END
```

- **IDS_APP_TITLE**

– Window title string.

- **IDC_SPIWAVWIN32**

– Application name (“SPISOCKETSWIN32”) used by LoadString calls.

## Resource Identifiers (resource.h)

**File:** resource.h

All identifiers generated by Visual C++ and used in the RC file:

| Identifier | Value | Description |
| --- | --- | --- |
| IDC_MYICON | 2 | Unused default icon identifier |
| IDD_SPIWAVWIN32_DIALOG | 102 | (Unused) dialog template from AppWizard |
| IDS_APP_TITLE | 103 | String table entry for window title |
| IDD_ABOUTBOX | 103 | About dialog identifier |
| IDM_ABOUT | 104 | “About” command menu item |
| IDM_EXIT | 105 | “Exit” command menu item |
| IDI_SPIWAVWIN32 | 107 | Large application icon |
| IDI_SMALL | 108 | Small icon for window title |
| IDC_SPIWAVWIN32 | 109 | Menu and string table resource name |
| IDR_MAINFRAME | 128 | Main frame identifier (e.g., toolbar/resource) |
| IDC_STATIC | –1 | Static control (no identifier) |


Next defaults for new resources (managed by App Studio):

This mapping between resource script entries and their numeric IDs allows you to customize or extend the UI—for example, adding new menu commands or dialogs—by editing **spisocketswin32.rc** and **resource.h** accordingly.
